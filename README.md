# URL Shortener
A simple URL shortener made to exercise Kubernetes deployment.

## Features
* Create, edit, delete short links.
* Choose a path prefix for your short links.
* Protect short links with a password.

## Development
Requires:
1. Python 3.10.2
2. Poetry 1.4.1
3. Task 3.13.0
